온라인 랭킹 설정 방법
Firebase Realtime Database를 하나 만들고, 그 주소를 이 게임에 넣으면 각자 스마트폰에서 같은 랭킹을 볼 수 있어.
1단계 Firebase 콘솔에서 Realtime Database를 만들기
2단계 Database URL 복사하기
3단계 이 게임의 온라인 연결 설정에 URL 붙여넣기
4단계 친구도 같은 파일과 같은 URL을 사용하기
5단계 모두 같은 그룹코드를 저장하기
임시 테스트용 규칙 예시
테스트는 쉽지만 공개 쓰기 상태라서, 나중에는 규칙을 꼭 더 안전하게 바꾸는 게 좋아.
입력 예시
DB URL: https://내프로젝트-default-rtdb.asia-southeast1.firebasedatabase.app
토큰: 비워도 되고, 규칙상 필요할 때만 입력
그룹코드: SUBIN
